Backpropagation neural network. See the demo applications for working examples.

In basic terms, the task is to define a network (i.e. node populations) and then train it. Once trained, you can then submit patterns to the network and obtain a response from it.

BackProp is the main entry point and contains methods to define and invoke the NN.

NetWork is a serialized BP network.

Arc/Node, etc are network elements.
